# Cài đặt
## Tạo project
```
vue create vue-3-template
```
Chọn: __Default (Vue 3) ([Vue 3] babel, eslint)__
## Thêm typescript
Tham khảo: https://v3.vuejs.org/guide/typescript-support.html#augmenting-types-for-globalproperties
```
vue add typescript
```
